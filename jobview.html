Software Engineer - Navision, .NET from CyberCoders

Overland Park, KS 66062

About the Job

We are fast growing, extremely well funded and now established in the health care industry platform and data analysis space. We are backed by huge partners including significant investment by Google Ventures.

We are seeking a talented Senior Software Engineer who is interested in pushing their boundaries and taking on a role that involves significant work in an environment that relies on systems for ingesting, monitoring, and visualizing massive amounts of streaming sensors.

Our team uses technologies that include Microsoft Dynamics NAV (Navision) C/AL or C/SIDE coding experience required.

If you are a Software Developer and you like solving problems, finding and fixing issues, improving application performance and contributing to high transaction, please read on.

Top Reasons to Work with Us

- Generous compensation and medical benefits package
- Work with modern cutting edge technologies
- Leverage the performance, power, and scalability of modern cloud computing best practices

What You Will Be Doing

- Support and integrate third party solutions such as Microsoft Dynamics NAV and Sales Force.Participate in requirements analysis and detail design of new systems and functionality.
- Translate business requirements into intuitive software solutions to support organizational efficiencies.
- Collaborate with fellow software developers while building a solution knowledge base.
- Participant within an Agile (Scrum) development team.
- Develop and test functional additions to acceptable tolerances.

What You Need for this Position

- Microsoft Dynamics NAV (Navision) C/AL or C/SIDE coding experience required.
- 2+ years applications development.
- 2+ years enterprise class application architecture required.
- 2+ years .NET and MS SQL Server experience required.
- 3+ years HTML, CSS, JavaScript required.
- XML design and development required.

What's In It for You

-Work on extremely challenging and thought provoking problems
- Great base salary + equity + bonus + PTO and great other benefits too!
So, if you are a Software Developer with strong Microsoft Dynamics NAV (Navision) experience, please apply today!
